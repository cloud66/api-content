Returns a paginated collection of Helm releases within the specified formation.

Helm releases manage the lifecycle of applications deployed via Helm charts in Kubernetes environments. Each release represents an installed instance of a Helm chart.

<Callout type="info" title="Feature Flag Required">
This endpoint requires the formations feature flag to be enabled on your account. Contact support@cloud66.com if you need access.
</Callout>
