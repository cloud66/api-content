Updates an existing Helm release with new configuration settings.

Modifies the specified Helm chart deployment with new values, chart version, or other configuration changes. This operation requires redeploy permissions and validates the Helm configuration before applying updates.

<Callout type="warning" title="Feature Flag Required">
This endpoint requires the formations feature flag to be enabled on your account. Contact support@cloud66.com if this feature is not available.
</Callout>

<RelatedModels models={['HelmRelease', 'Formation', 'Stack']} />

**Update Capabilities:**
- **Chart Version**: Upgrade or downgrade to different chart versions
- **Values Configuration**: Modify Helm values YAML for customization
- **Display Name**: Update the release display name
- **Repository**: Change the Helm chart repository (if supported)

<Callout type="info" title="Permissions Required">
You must have helm release edit rights on the formation to update releases.
</Callout>
