Retrieves detailed information about a specific deployment policy.

Returns comprehensive details about the specified policy, including the complete configuration body, version history, and metadata for deployment rule management.

<Callout type="warning" title="Feature Flag Required">
This endpoint requires the formations feature flag to be enabled on your account. Contact support@cloud66.com if this feature is not available.
</Callout>

<RelatedModels models={['Policy', 'Formation', 'Stack']} />

**Policy Details:**
- **Configuration Body**: Complete YAML/JSON policy definition
- **Sequence Order**: Execution priority during deployments
- **Version Tracking**: Policy version and modification history
- **Tag Management**: Organization and categorization labels
- **Access Control**: User permissions and modification tracking

<Callout type="info" title="Permission Required">
You must have policy view rights on the formation to access policy details.
</Callout>

**Common Policy Types:**
- **NetworkPolicy**: Kubernetes network security rules
- **LimitRange**: Resource consumption constraints
- **PodSecurityPolicy**: Pod security configurations
- **ResourceQuota**: Namespace resource limits
- **PriorityClass**: Pod scheduling priorities
