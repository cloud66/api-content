Retrieves comprehensive pipeline information for a specific formation, including deployment configuration and workflow details.

Returns the complete pipeline structure showing how the formation will be deployed, including snapshot data, workflow steps, Helm releases, and execution context.

<RelatedModels models={['Pipeline', 'Formation', 'Snapshot', 'Workflow', 'HelmRelease']} />

**Pipeline Components:**
- **Formation**: The target deployment formation
- **Snapshot**: Code snapshot to deploy (latest if not specified)
- **Workflow**: Deployment workflow steps and configuration
- **Helm Releases**: Associated Helm chart deployments
- **Execution Context**: User and timing information

<Callout type="info" title="Default Behavior">
If no snapshot is specified, the pipeline uses the latest available snapshot. If no workflow is specified, the formation's default workflow is used.
</Callout>

**Query Options:**
- **snapshot_uid**: Use a specific snapshot (or "latest" for most recent)
- **use_latest**: Boolean flag to force using latest snapshot data
- **workflow**: Specify a particular workflow by name
